# Phase 02: 分析与复杂度评级

> **阶段代号**: P2  
> **入口条件**: P1 已完成  
> **预计耗时**: 小型项目 30–60 分钟 · 中型 1–2 小时 · 大型 2–4 小时

---

## 目标

对 P1 产出的每个模块，基于证据进行复杂度评级——结合客观指标与结构化主观推理。

---

## 子阶段

### P2a: 模块独立评级 [可并行]

对每个模块独立执行：

1. 采集客观指标
2. 结构化主观推理
3. 综合得出复杂度评级

完成后更新各模块 L1 报告，写入评级结果。

### P2b: 依赖关系整体分析 [串行]

跨模块依赖分析。需要所有模块的 L1 报告作为输入。完成后更新总体报告中的模块依赖图。

---

## 执行

加载并执行 [Skill 02 — 复杂度评级](../skills/skill-02-rate.md)。

参考定义：
- [complexity_levels — 复杂度等级定义](../definitions/complexity_levels.md)
- [objective_metrics — 客观指标定义](../definitions/objective_metrics.md)

---

## 验收标准

- [ ] 所有 L1 报告均已包含复杂度评级及结构化推理
- [ ] 评级为 **高** 或 **极高** 的模块已附带拆分评估
- [ ] 总体报告中的模块依赖图已更新
- [ ] 客观指标下限已检查通过
