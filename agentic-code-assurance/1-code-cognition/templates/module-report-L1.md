# 模块报告模板（L1 — 迭代阶段）

> **用途**：P1→P2→P3 迭代循环期间的轻量级模块报告，聚焦模块识别与复杂度评级
> **位置**：`docs/codearch/modules/<module_name>.md`（迭代阶段；收敛后由 P4 升级为 L2）

---

## 使用说明

1. 在 P1→P2→P3 迭代过程中使用，每个模块一份独立文件。
2. 轻量级：重点完成模块识别、职责描述、依赖概览和复杂度评级。
3. 深度分析（关键设计要点、数据流、生命周期等）推迟到 P4 阶段补充。
4. 复杂度评级时参阅 [complexity_levels.md](../definitions/complexity_levels.md)。
5. 收敛后所有 L1 报告将在 P4 阶段升级为 L2 完整版。

---
