# 模块报告模板（L2 — 完整版）

> **用途**：P4 阶段深度分析后的完整模块报告，包含 Phase 2 所需的全部字段
> **位置**：`docs/codearch/modules/<module_name>.md`

---

## 使用说明

1. 所有模块在 P4 阶段从 L1 升级为 L2，补充深度分析字段。
2. 包含 Phase 2 所需的全部信息（代码特征、数据流、生命周期、并发不变量等）。
3. 父模块（经拆分的模块）在末尾添加「子模块索引」章节。
4. 复杂度评级时参阅 [complexity_levels.md](../definitions/complexity_levels.md)；验证时参阅 [validation_levels](../definitions/validation_levels.md)。
5. 「信息来源」记录分析依据；「使用示例」至少填写一个基本用法；「验证状态」复杂度 ≥ 高时必填。

> **注意**：本模板包含 L1 阶段已有的字段（标注 **[L1]**）和 P4 深度分析新增字段（标注 **[P4 新增]**）。
