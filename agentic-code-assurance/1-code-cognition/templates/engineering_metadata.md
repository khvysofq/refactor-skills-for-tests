# 工程元数据模板

> **用途**：由 P0 阶段机械化采集，供后续阶段引用
> **位置**：`docs/codearch/engineering_metadata.md`

---

## 使用说明

1. P0 阶段机械化填写所有章节，不做主观判断，仅采集客观事实。
2. P1/P2 阶段引用本文档中的数据（目录结构、构建 Target、代码行数等）作为分析输入。
3. 本文档不直接被 Phase 2/3 消费，仅作为 Phase 1 内部数据源。
